# রোজা রাখার নিয়ত:
উচ্চারণ: নাওয়াইতু আন আছুমা গাদাম, মিন শাহরি রমাদানাল মুবারাক; ফারদাল্লাকা ইয়া আল্লাহু, ফাতাকাব্বাল মিন্নি ইন্নিকা আনতাস সামিউল আলিম।  
অর্থ: হে আল্লাহ! আমি আগামীকাল পবিত্র রমজানের তোমার পক্ষ থেকে নির্ধারিত ফরজ রোজা রাখার নিয়্যত করলাম।  

# ইফতারের দোয়া:  
আল্লাহুম্মা লাকা ছুমতু ওয়া আলা রিযক্বিকা ওয়া আফতারতু বিরাহমাতিকা ইয়া আরহামার রাহিমিন।  
অর্থ: হহে আল্লাহ! আমি আপনার উদ্দেশে রোজা রেখেছি এবং আপনার দেওয়া রিজিক দিয়ে ইফতার করছি । আপনি আমার রোজা কবুল করুন।। 

# জানাজা
আল্লাহুম্মাগ ফিরলি হায়্যিনা ওয়া মাইয়্যিতিনা ওয়া শাহিদিনা ওয়া গায়িবিনা ওয়া সাগীরিনা ওয়া কাবীরিনা ওয়া যাকারিনা ওয়া উনছানা। আল্লাহুম্মা মান আহইয়াহতাহু মিন্না, ফাআহয়িহী আলাল ইসলামি ওয়া মান তাওয়াফ ফাইতাহু মিন্না ফাতাওয়াফফাহ আলাল ইমান।
