# জানাজা
আল্লাহুম্মাগ ফিরলি হায়্যিনা ওয়া মাইয়্যিতিনা ওয়া শাহিদিনা ওয়া গায়িবিনা ওয়া সাগীরিনা ওয়া কাবীরিনা ওয়া যাকারিনা ওয়া উনছানা। আল্লাহুম্মা মান আহইয়াহতাহু মিন্না, ফাআহয়িহী আলাল ইসলামি ওয়া মান তাওয়াফ ফাইতাহু মিন্না ফাতাওয়াফফাহু আলাল ইমান।

# রোগ থেকে মুক্তির দোয়া
আল্লাহুম্মা ইন্নি আওযুবিকা মিনাল বারাসি ওয়াল জুনুনি ওয়াল জুজামি ওয়া মিন সাইয়্যিল আসকাম
